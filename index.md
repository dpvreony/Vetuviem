## Welcome to Vetuviem

Vetuviem is a DSL toolset for producing reusable View to ViewModel binding logic for ReactiveUI. The UI framework support status is as follows

UI Framework | Base Object | Status
--- | --- | ---
WinForms | System.Windows.Forms.Control | Not Supported yet
WinUI | Microsoft.UI.Xaml.UIElement | Not Supported yet
WPF | System.Windows.UIElement | Alpha
Xamarin Forms | ? | Not Supported yet
